# QuickDonate Overview

QuickDonate is a one-time donation plugin for WordPress.

Implemented features:

- Popup donation form via shortcode
- Preset amounts
- Optional custom amount
- Donor email capture
- Server-side Paystack verification
- Donation logs in wp-admin
- Thank-you message after verified success
- Optional thank-you email after verified success
- Gateway-ready internal structure

Current gateway support:

- Paystack: fully supported
- Additional gateways: not implemented yet
