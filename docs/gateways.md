# Gateways

QuickDonate is structured so payment handling can be expanded beyond Paystack.

Current implementation:

- `QuickDonate_Gateway_Interface`
- `QuickDonate_Paystack_Gateway`

Current working gateway:

- Paystack

Future-ready direction:

- Stripe
- Flutterwave
- Manual payments

These future gateways are not implemented in the current release.
