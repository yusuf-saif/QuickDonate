# Configuration

## General Settings

- Enable or disable donations
- Set button label
- Set donation currency
- Configure the thank-you message
- Optionally select success and failure redirect pages

## Donations

- Set preset donation amounts as comma-separated values
- Enable or disable custom amount entry
- Set minimum amount
- Set maximum amount

## Emails

- Enable thank-you email
- Set sender name and sender email
- Set email subject and body

Supported placeholders:

- `{amount}`
- `{currency}`
- `{email}`
- `{reference}`
- `{site_name}`

## Advanced

- Choose test or live mode
- Enter Paystack public and secret keys
