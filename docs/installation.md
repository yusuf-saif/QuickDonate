# Installation

1. Upload the plugin to `/wp-content/plugins/`.
2. Activate `QuickDonate` in WordPress.
3. Open the `QuickDonate` admin menu.
4. Set your Paystack test or live keys.
5. Configure currency, amounts, and email settings.
6. Add `[quickdonate_popup]` to a page or post.

Legacy shortcodes still work for existing pages:

- `[paystack_donation_popup]`
- `[quickgive_donation_popup]`
